Authentification Standard

Connexion sécurisée via nom d'utilisateur et mot de passe pour les utilisateurs standards et administrateurs.

  • Utilisateurs standards: Accès basique interfaces
  • Administrateurs: Gestion modules attribués
  • Super Admin: Accès complet toutes interfaces
  • Mot de passe: Politique complexité exigée
  • Session: Expiration après inactivité

Double Authentification (2FA)

Authentification renforcée pour les accès sensibles des Assistants Admin.

  • SMS: Code envoyé au mobile enregistré
  • Email: Code envoyé à l'adresse email
  • App authenticator: Google/Microsoft Authenticator
  • Validité: Code valable 5 minutes
  • Bloquage: Après 3 tentatives erronées

Gestion des Rôles et Permissions

Accès définis selon le rôle hiérarchique de chaque utilisateur.

  • Super Admin: Accès complet + gestion utilisateurs
  • Admin/Assistant UBS: Accès complet UBS + lecture autres
  • Admin/Assistant C2A: Accès complet C2A + lecture autres
  • Admin/Assistant SITE: Accès complet SITE + lecture autres
  • Admin ACR: Accès complet ACR + lecture autres
